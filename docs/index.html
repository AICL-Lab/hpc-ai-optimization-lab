Modules

Seven Kernel Modules

Each module focuses on a specific CUDA optimization scenario, from basic elementwise to experimental CUDA 13 features.

01 · Elementwise

ReLU, Sigmoid, Transpose. Learn coalesced access, vectorization, and bank conflict avoidance.

src/01_elementwise · Stable

02 · Reduction

Softmax, LayerNorm, RMSNorm. Master warp shuffle, block reduction, and online algorithms.

src/02_reduction · Stable

03 · GEMM

7-step matrix multiplication optimization from naive to Tensor Core, benchmarked against cuBLAS.

src/03_gemm · Stable

04 · Convolution

Implicit GEMM convolution implementation. Winograd path uses GEMM fallback.

src/04_convolution · Stable

05 · Attention

FlashAttention forward with online softmax. Supports float precision, head_dim=64.

src/05_attention · Stable

06 · Quantization

INT8 per-row quantization/dequantization, FP8 scaling utilities.

src/06_quantization · Stable

07 · CUDA 13 Features

Experimental TMA, Thread Block Clusters, FP8 GEMM with fallback implementations.

src/07_cuda13_features · Experimental
Roadmap

Learning Path

Three stages from memory basics to cutting-edge architecture features.

Beginner · 1-2 weeks

Memory & Reduction Fundamentals

  • Memory Optimization → Coalescing, vectorization, shared memory
  • Reduction Optimization → Block-level reduction, warp shuffle
  • GEMM Steps 1-4 → Naive to shared memory tiling
Intermediate · 2-4 weeks

Tensor Core & Attention

  • GEMM Steps 5-7 → Tensor Core WMMA/MMA, register tiling
  • FlashAttention → IO-aware design, online softmax
Advanced · Ongoing

Cutting-Edge Features

  • CUDA 13 → TMA, Clusters, FP8 (experimental)
  • CUTLASS source code deep dive
  • Latest research papers and toolchain updates

ncu --set roofline -o roofline ./app

Build Commands

# Standard build
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)
ctest --test-dir build --output-on-failure

# With Python bindings
cmake -S . -B build -DBUILD_PYTHON_BINDINGS=ON
cmake --build build
export PYTHONPATH="$(pwd)/build/python:$PYTHONPATH"

Python API Example

import torch
import hpc_ai_opt

# Create tensors
x = torch.randn(1024, 1024, device="cuda")
y = torch.empty_like(x)

# Use optimized CUDA kernel
hpc_ai_opt.elementwise.relu(x, y)

# Verify
torch.testing.assert_close(y, torch.relu(x))
